# LAD.Py - Sistema de Votação Digital
# Descrição do projeto
Projeto integrador I da disciplina de Engenharia de Software da PUC Campinas. O Sistema LAD.Py consiste no desenvolvimento de um sistema de votação digital fictício, executado via terminal, com foco em Python, MySQL, e segurança da informação

- MySQL Workbench
- MYSQL Server
- mysql.connector
- Git e Github
- Github Projects
- VScode
# Estrutura Inicial do Projeto
- main.py - onde o projeto será rodado
- menu.py - todos os menus
- conexaobd.py - conexão com o banco
- banco.sql - hospedagem do banco e criação de tabelas
# Como executar
1. Clonar o repositório
2. Criar o banco de dados MYSQL
3. Instalar a biblioteca mysql-connector-python usando pip install mysql-connector-python em um terminal
4. Executar main.py em um terminal
